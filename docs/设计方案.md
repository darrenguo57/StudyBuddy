# 儿童作业智能监督与回顾系统 — 完整设计方案

## 一、项目概述

### 1.1 项目名称
**StudyBuddy — 智能作业陪伴与回顾系统**

### 1.2 核心目标
通过笔记本摄像头 + AI 视觉 + 语音交互，实现：
- **实时坐姿监督与纠正**：自动检测驼背、歪头、距离过近等问题并语音提醒
- **作业过程完整记录**：全程录像，保留原始素材
- **一键生成倍速回顾视频**：作业完成后自动剪辑，生成带时间轴标注的加速回顾
- **智能评分与总结**：基于专注度、坐姿合规率、完成效率生成综合报告
- **月度积分兑换奖励**：将得分转化为可兑现的奖励，形成正向激励闭环

### 1.3 适用场景
- 6~15 岁儿童/青少年家庭作业监督
- 远程家长无法实时陪伴的场景
- 培养自主学习习惯与良好坐姿

## 三、技术选型与开源库对比

### 3.1 人体姿态检测

| 方案 | Stars | 关键点数量 | 性能(FPS) | 依赖 | 推荐度 |
|------|-------|-----------|-----------|------|--------|
| **MediaPipe Pose** | 28k+ | 33 个 3D | 30+ (CPU) | 轻量 pip install | ★★★★★ |
| OpenPose | 31k+ | 25/135 个 | 5-10 (CPU) | C++ 编译复杂 | ★★★☆ |
| YOLOv8-Pose | 30k+ | 17 个 2D | 20+ (CPU) | 需 GPU 最佳 | ★★★☆ |
| MoveNet | 7k+ | 17 个 2D | 30+ (CPU) | TensorFlow Lite | ★★★★ |

**最终选择：MediaPipe Pose**
- 理由：纯 CPU 即可达到 30+ FPS，33 个 3D 关键点精度足够，安装极简，社区生态成熟
- 关键点选取：NOSE、LEFT/RIGHT_EAR、LEFT/RIGHT_SHOULDER、LEFT/RIGHT_HIP 共 7 个核心点

### 3.2 视频录制

| 方案 | 优点 | 缺点 | 推荐度 |
|------|------|------|--------|
| **OpenCV VideoWriter** | 与姿态检测同框架、无额外依赖、支持多编码器 | 仅支持固定帧率录制 | ★★★★★ |
| imageio-ffmpeg | 灵活性强 | 需额外配置 | ★★★☆ |
| PyAV | 性能好 | 文档少 | ★★★☆ |

**最终选择：OpenCV cv2.VideoWriter**（与姿态检测流水线天然集成）

### 3.3 视频自动剪辑与倍速

| 方案 | 功能覆盖 | 性能 | 文档质量 | 推荐度 |
|------|---------|------|---------|--------|
| **MoviePy** | 裁剪/拼接/倍速/字幕/特效/音频混合 | 好(基于FFmpeg) | ★★★★★ | ★★★★★ |
| FFmpeg 命令行 | 全部 | 最优 | 命令行繁琐 | ★★★★ |
| ffmpeg-python | 中等 | 好 | ★★★☆ | ★★★☆ |
| PyAv | 底层操作 | 好 | ★★☆ | ★★☆ |

**最终选择：MoviePy（主）+ FFmpeg（辅助）**

MoviePy 核心能力映射本项目需求：

| 需求 | MoviePy 实现 |
|------|-------------|
| 倍速加速 | `clip.fx(vfx.speedx, factor=4)` |
| 片段裁剪拼接 | `concatenate_videoclips([c1, c2, c3])` |
| 字幕叠加(违规时间标注) | `CompositeVideoClip + TextClip` |
| 片头片尾 | 静态图片 + 文字 + 转场 |
| 背景音乐合成 | `CompositeAudioClip` 叠加 |
| 视频导出 | `clip.write_videofile()` 支持 H.264 |

### 4.2 坐姿检测模块 `posture_detector.py`

#### 4.2.1 检测维度与算法

| 检测项 | 关键点组合 | 判断逻辑 | 阈值 | 违规等级 |
|--------|-----------|---------|------|---------|
| **头部前倾(驼背)** | NOSE → SHOULDER 中点 | shoulder_mid → nose 向量角度 > 阈值 | 25° | 🔴 严重 |
| **头部左/右歪** | NOSE, LEFT_EAR, RIGHT_EAR | 两耳连线与水平夹角 | ±12° | 🟡 轻微 |
| **身体左/右倾斜** | LEFT_SHOULDER, RIGHT_SHOULDER | 两肩连线与水平夹角 | ±10° | 🟡 轻微 |
| **距离过近** | NOSE.z | z 轴深度值 < 阈值 | z < -0.15 | 🔴 严重 |
| **趴桌检测** | NOSE.y vs SHOULDER_MID.y | 鼻子 y 坐标低于肩膀中点 | y_nose > y_shoulder | 🔴 严重 |

#### 4.2.2 违规记录数据结构

```python
@dataclass
class PostureViolation:
    timestamp: float          # 视频时间戳(秒)
    violation_type: str       # 'head_forward' | 'head_tilt' | 'body_tilt' | 'too_close' | 'lying_down'
    severity: str             # 'warning' | 'critical'
    frame_snapshot: bytes     # 违规帧截图(可选)
```

#### 4.2.3 防抖与提醒策略

```
状态机设计：
  ┌──────────┐    连续3帧违规    ┌──────────┐   持续5秒违规    ┌──────────┐
  │ NORMAL   │ ───────────────→ │ WARNING  │ ───────────────→ │ CRITICAL │
  │          │                  │ (仅记录) │                  │ (语音提醒)│
  └────┬─────┘                  └────┬─────┘                  └────┬─────┘
       ↑                             │                             │
       │      连续10帧正常           │      连续2秒正常             │
       └─────────────────────────────┴─────────────────────────────┘

提醒规则（防骚扰）：
  - 轻度违规（歪头/倾斜）: 警告状态不提醒，仅 CRITICAL 状态提醒
  - 严重违规（驼背/趴桌）: 进入 CRITICAL 立即提醒
  - 提醒冷却: 同类型违规至少间隔 30 秒
  - 单次作业最多提醒 10 次
  - 提醒语音示例（pyttsx3 合成）：
    "小朋友，把头抬起来一点哦~"
    "注意坐直，不要趴在桌子上"
    "距离屏幕太近了，往后靠一靠"
```

### 4.4 视频自动剪辑模块 `video_editor.py`

#### 4.4.1 处理流程

```
原始视频(15FPS 全纪录)
    │
    ├─ Step 1: 片段分析
    │   ├─ 加载 posture_events 时间线
    │   ├─ 识别"有效写作业段"（持续坐姿合规 > 30 秒的连续段）
    │   ├─ 识别"违规段"（标记为红色标注）
    │   └─ 识别"起身/离开段"（无人检测 → 跳过或加速至 16x）
    │
    ├─ Step 2: 速度映射
    │   ├─ 有效段: 4x 加速（保留细节）
    │   ├─ 违规段: 2x 加速（让家长看清问题）
    │   └─ 离开段: 直接裁剪掉
    │
    ├─ Step 3: 标注叠加
    │   ├─ 左上角: 实时计时器 "已写作业 25:30"
    │   ├─ 违规时刻: 红色半透明条 + 违规类型文字
    │   └─ 底部: 坐姿合规率进度条 (绿=达标 / 黄=警告 / 红=违规)
    │
    ├─ Step 4: 片头片尾
    │   ├─ 片头(3秒): 日期 + 科目名称 + 可爱动画
    │   └─ 片尾(5秒): 综合评分面板 + 各项得分
    │
    └─ Step 5: 合成输出
        ├─ 背景音乐: 轻快纯音乐(可选，默认静音)
        ├─ 输出格式: MP4 H.264 1080p @30fps
        └─ 命名: YYYY-MM-DD_科目_回顾.mp4
```

#### 4.4.2 MoviePy 核心代码骨架

```python
from moviepy.editor import VideoFileClip, concatenate_videoclips, CompositeVideoClip, TextClip
from moviepy.video.fx import speedx
import moviepy.video.fx.all as vfx

def generate_review_video(raw_video_path, events, output_path, session_data):
    raw = VideoFileClip(raw_video_path)

    clips = []
    for segment in analyze_segments(events, session_data.duration):
        sub = raw.subclip(segment.start, segment.end)

        # 变速
        if segment.type == 'effective':
            sub = sub.fx(vfx.speedx, factor=4)
        elif segment.type == 'violation':
            sub = sub.fx(vfx.speedx, factor=2)
            # 叠加违规标注
            label = TextClip(f"⚠ {segment.violation_type}", fontsize=24, color='red')
            sub = CompositeVideoClip([sub, label.set_position(('center', 'top'))])
            label.close()
        else:  # away
            sub = sub.fx(vfx.speedx, factor=16)

        clips.append(sub)

    final = concatenate_videoclips(clips, method="compose")

    # 叠加计时器
    timer_overlay = create_timer_overlay(session_data.duration)
    final = CompositeVideoClip([final, timer_overlay])

    # 合成片头片尾
    intro = create_intro_clip(session_data)
    outro = create_outro_clip(session_data)
    final = concatenate_videoclips([intro, final, outro])

    final.write_videofile(output_path, fps=30, codec='libx264', bitrate='2000k')
```

### 4.5 评分引擎模块 `scoring_engine.py`

#### 4.5.1 评分维度与权重

| 评分维度 | 满分 | 权重 | 计算方式 |
|---------|------|------|---------|
| **坐姿合规率** | 100 | 40% | 合规帧数 / 总检测帧数 × 100 |
| **专注时长** | 100 | 25% | 连续合规段总时长 / 总时长 × 100 |
| **作业完成效率** | 100 | 20% | 由家长设定预期时间，实际时间与预期比值评分 |
| **违规纠正响应** | 100 | 15% | 提醒后 10 秒内纠正的违规次数 / 总提醒次数 × 100 |

**综合得分 = Σ(维度得分 × 权重)**

#### 4.5.2 扣分规则（过程监督减分点）

| 违规事件 | 扣分 | 详细说明 |
|---------|------|---------|
| 头部前倾/驼背（持续 > 5 秒） | -3 分/次 | 语音提醒后 10 秒内未纠正再扣 2 分 |
| 趴桌 | -5 分/次 | 严重违规，一次扣 5 分，不纠正再扣 5 分 |
| 头部歪斜（持续 > 10 秒） | -1 分/次 | 累计超过 5 次开始加倍扣分 |
| 身体倾斜（持续 > 10 秒） | -1 分/次 | 同上 |
| 距离过近 | -2 分/次 | 语音提醒后 10 秒内未纠正再扣 2 分 |
| 起身离开（单次 > 3 分钟） | -5 分/次 | 合理休息(1~3分钟离开)不扣分 |
| 作业中途结束（总时长 < 10 分钟） | -10 分 | 防止恶意刷分 |
| 频繁晃动/多动（帧间位移 > 阈值） | -2 分/次 | 连续 5 秒晃动算 1 次 |

#### 4.5.3 加分规则

| 加分事件 | 加分 | 详细说明 |
|---------|------|---------|
| 连续 30 分钟零违规 | +5 分 | 每 30 分钟可叠加 |
| 连续 5 次作业无严重违规 | +10 分 | 进步奖励 |
| 提前完成作业（效率 > 120%） | +5 分 | 鼓励高效 |
| 单次作业 > 60 分钟 | +3 分 | 毅力奖励 |
| 违规后 5 秒内自行纠正（未被提醒） | +2 分/次 | 自觉性奖励 |

#### 4.5.4 最终评级

| 综合得分 | 评级 | 图标 |
|---------|------|------|
| 95-100 | S 级 — 超级学霸 | ⭐⭐⭐ |
| 85-94  | A 级 — 优秀表现 | ⭐⭐ |
| 70-84  | B 级 — 良好 | ⭐ |
| 60-69  | C 级 — 需要改进 | — |
| < 60   | D 级 — 要加油哦 | — |

## 五、奖励兑换机制

### 5.1 积分体系

```
月度总积分 = Σ(每次作业综合得分) + 额外奖励分

额外奖励分：
  ├─ 全勤奖: 当月作业天数 ≥ 20 天 → +50 分
  ├─ 进步奖: 当月平均分较上月提升 ≥ 10 分 → +30 分
  ├─ 连续达标: 连续 10 天评级 B 以上 → +20 分
  └─ 周末挑战: 周末完成额外学习任务 → +15 分/次
```

### 5.2 奖励阶梯

| 积分门槛 | 奖励内容 | 说明 |
|---------|---------|------|
| 100 分 | 选择晚餐菜单权 | 低门槛正向反馈 |
| 200 分 | 30 分钟额外游戏时间 | 中等奖励 |
| 350 分 | 周末户外活动 | 亲子互动 |
| 500 分 | 购买一本喜欢的书 / 小玩具(≤50元) | 月度大奖 |
| 800 分 | 月度特别大礼(≤100元) | 需要多个月积累 |

### 5.3 家长控制面板

```
家长可配置：
  ├─ 奖励内容自定义（文字 + 图片 + 预算）
  ├─ 积分门槛调整
```

## 九、关键技术难点与对策

| 难点 | 风险 | 对策 |
|------|------|------|
| 坐姿检测误判（孩子正常活动被误判违规） | 频繁误提醒影响体验 | 防抖策略（连续 N 帧才确认）+ 家长可调阈值 |
| 低光照环境下检测精度下降 | 晚上作业时关键点丢失 | 自动亮度检测 → 提示开灯 / 红外摄像头 |
| 视频文件过大 | 1280×720@15fps 约 1.5GB/小时 | 录制分辨率降为 960×540 + 原始视频可选"仅保留回顾视频" |
| 孩子故意遮挡摄像头 | 坐姿检测失效 | 人脸丢失 > 10 秒 → 语音提示"看不到你了哦" |
| 多孩共用一台电脑 | 关键点模型不同 | 支持"切换孩子"配置文件，独立评分 |
