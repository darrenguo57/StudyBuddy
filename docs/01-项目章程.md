# 项目章程 — StudyBuddy 智能作业监督系统

## 1. 项目基本信息
- 项目名称：StudyBuddy — 智能作业陪伴与回顾系统
- 项目编号：SB-2026-001
- 项目经理：Marvis AI
- 发起日期：2026-06-21
- 目标交付日期：2026-08-02（6周）
- 版本：v1.0

## 2. 项目目标
构建一个基于计算机视觉的儿童作业智能监督系统，通过笔记本摄像头实现：
- 实时坐姿检测与语音纠正
- 作业过程完整录制
- 一键生成倍速回顾视频（含违规标注）
- 智能评分与月度积分兑换奖励机制

## 3. 项目范围
### 范围内
- 摄像头管理（双线程采集+录制）
- MediaPipe 姿态检测（5项坐姿检测）
- pyttsx3 语音提醒
- MoviePy 倍速视频剪辑
- 评分引擎（4维加权评分）
- SQLite 数据持久化
- PyQt6 桌面 UI
- 月度积分与奖励兑换

### 范围外
- 移动端 APP
- 云端同步
- AI 作业内容批改
- 多人同时监督
