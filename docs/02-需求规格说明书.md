# 需求规格说明书 (SRS) — StudyBuddy

## 1. 引言
### 1.1 目的
本文档定义 StudyBuddy 系统的功能与非功能需求。

### 1.2 范围
适用于 v1.0 版本所有模块。

### 1.3 定义与缩略语
- SRS: Software Requirements Specification
- FPS: Frames Per Second
- TTS: Text-to-Speech

## 2. 总体描述
### 2.1 产品视角
StudyBuddy 是一个独立的桌面应用程序，运行在 Windows 10/11 笔记本上，依赖内置/外接摄像头。

### 2.2 用户角色
| 角色 | 描述 |
|------|------|
| 孩子（主用户） | 6-15岁，使用系统进行作业监督 |
| 家长（管理员） | 配置参数、审批奖励、查看报告 |

### 2.3 运行环境
- OS: Windows 10/11 64位
- Python: 3.10+
- 摄像头: 内置或 USB 外接
- 存储: ≥10GB 可用空间
- 内存: ≥4GB

## 3. 功能需求

### FR-01 摄像头管理
- FR-01-01: 系统启动时自动检测并连接摄像头
- FR-01-02: 支持 1280×720 @30FPS 实时预览
- FR-01-03: 支持 15FPS 视频录制（H.264 编码）
- FR-01-04: 点击"开始"启动录制，"结束"停止录制
- FR-01-05: 录像保存至 recordings/ 目录，命名格式 YYYY-MM-DD_HH-MM-SS_raw.mp4

### FR-02 坐姿检测
- FR-02-01: 实时检测 5 项坐姿指标（头部前倾、歪头、身体倾斜、距离过近、趴桌）
- FR-02-02: 检测结果在 UI 上用绿/黄/红指示灯展示
- FR-02-03: 违规事件记录时间戳和类型
- FR-02-04: 防抖机制：连续 3 帧确认违规，持续 5 秒触发提醒
- FR-02-05: 同类型违规提醒间隔 ≥30 秒

### FR-03 语音交互
- FR-03-01: 作业开始时语音引导
- FR-03-02: 违规时语音提醒（亲切鼓励语气）
- FR-03-03: 每 15 分钟正向鼓励
- FR-03-04: 单次作业最多提醒 10 次
- FR-03-05: 离线和在线语音合成均支持（优先离线 pyttsx3）

### FR-04 视频剪辑
- FR-04-01: 作业结束后自动触发剪辑流程
- FR-04-02: 有效作业段 4x 加速（可配置）
- FR-04-03: 违规段 2x 加速（保留细节）
- FR-04-04: 离开段裁剪或 16x 加速
- FR-04-05: 叠加计时器、违规标注、合规率进度条
- FR-04-06: 生成片头（日期+科目）和片尾（评分面板）
- FR-04-07: 输出 1080p@30fps MP4

### FR-05 评分引擎
- FR-05-01: 四维度评分：坐姿合规率(40%)、专注时长(25%)、完成效率(20%)、纠正响应(15%)
- FR-05-02: 扣分规则：驼背-3、趴桌-5、距离过近-2、离开超3分钟-5
- FR-05-03: 加分规则：连续30分钟零违规+5、提前完成+5、自觉纠正+2
- FR-05-04: 等级评定：S(95+)、A(85-94)、B(70-84)、C(60-69)、D(<60)
- FR-05-05: 生成作业总结报告

### FR-06 数据管理
- FR-06-01: 所有数据存储在本地 SQLite 数据库
- FR-06-02: 支持查看历史作业记录和视频
- FR-06-03: 月度积分自动汇总

### FR-07 奖励机制
- FR-07-01: 积分阶梯：100/200/350/500/800 分对应不同奖励
- FR-07-02: 额外加分：全勤+50、进步+30、连续达标+20
- FR-07-03: 兑换流程：孩子发起 → 家长确认 → 积分扣除
- FR-07-04: 家长可自定义奖励内容和积分门槛

### FR-08 家长配置
- FR-08-01: 配置检测阈值（角度、距离等）
- FR-08-02: 配置提醒参数（间隔、最大次数）
- FR-08-03: 配置剪辑参数（倍速、码率）
- FR-08-04: 配置预期作业时长
- FR-08-05: 配置奖励阶梯

## 4. 非功能需求
### NFR-01 性能
- 坐姿检测 ≥20 FPS
- 视频录制不丢帧
- 剪辑速度 ≥实时 2x（40分钟视频在20分钟内完成剪辑）

### NFR-02 可用性
- 界面简洁，儿童可自主操作
- 语音提醒清晰可辨
- 一键开始/结束

### NFR-03 可靠性
- 连续运行 2 小时不崩溃
- 摄像头断开时优雅降级提示
- 异常退出时自动保存已录制的视频

### NFR-04 可维护性
- 模块化设计，各模块独立可测试
- 配置文件驱动，无需改代码即可调参

## 5. 附录
### A. 评分公式
综合得分 = 坐姿合规率×0.4 + 专注时长率×0.25 + 效率得分×0.2 + 纠正响应率×0.15
